# ✅ Requisitos Não Funcionais (RNF) – Sistema Kairos

| ID      | Nome                                               | Descrição                                                                                                   | Categoria   |
|---------|----------------------------------------------------|-------------------------------------------------------------------------------------------------------------|-------------|
| RNF01   | Interface Web Responsiva                           | A interface do sistema deve se adaptar a diferentes tamanhos de tela, incluindo smartphones e tablets.      | Usabilidade |
| RNF02   | Autenticação e Autorização por Tipo de Usuário     | O sistema deve permitir autenticação e restringir acesso com base no perfil: Administrador, Organizador ou Membro. | Segurança   |
| RNF03   | Segurança de Dados                                 | O sistema deve aplicar criptografia de senhas e controle de acesso a informações sensíveis.                 | Segurança   |
| RNF04   | Desempenho em Dispositivos Móveis                  | O sistema deve ser leve e otimizado para funcionar bem em celulares com recursos limitados.                 | Desempenho  |
