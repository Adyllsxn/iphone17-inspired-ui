# REQUISITO NÃO FUNCIONAL (RNF)

| ID      | Nome                                        | Descrição                                                                                               | Categoria     |
|---------|---------------------------------------------|-------------------------------------------------------------------------------------------------------|---------------|
| **RNF01** | Interface web responsiva                     | A interface do sistema será responsiva, adaptando-se a diferentes tamanhos de tela, incluindo dispositivos móveis. | Usabilidade   |
| **RNF02** | Suporte a autenticação e autorização por tipo de usuário | O sistema suportará autenticação de usuários, com autorização baseada no perfil (Administrador, Organizador, Membro). | Segurança     |
| **RNF03** | Segurança dos dados (criptografia de senhas, controle de acesso) | O sistema assegurará a segurança dos dados dos usuários, com criptografia de senhas e controle rigoroso de acesso às informações sensíveis. | Segurança     |
| **RNF04** | Sistema leve, otimizado para celulares      | O sistema será otimizado para garantir bom desempenho, especialmente em dispositivos móveis com recursos limitados. | Desempenho    |
| **RNF05** | Notificações automáticas programadas         | O sistema enviará notificações automáticas para os usuários, com base em eventos programados, sem intervenção manual. | Funcionalidade|
