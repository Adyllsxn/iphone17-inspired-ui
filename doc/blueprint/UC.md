# Casos de Uso - Sistema Kairos

# Tabela de Casos de Uso – Sistema Kairos

| Código | Nome do Caso de Uso               | Atores                             | Objetivo                                                    | Pré-condição                               | Pós-condição                                          |
|--------|-----------------------------------|------------------------------------|--------------------------------------------------------------|---------------------------------------------|--------------------------------------------------------|
| UC01   | Fazer Login                       | Membro, Organizador, Administrador | Permitir acesso ao sistema mediante autenticação.           | Usuário cadastrado e ativo.                | Sessão iniciada com permissões conforme o perfil.     |
| UC02   | Pedir Cadastro                    | Membro                             | Permitir que o usuário solicite criação de conta.           | Usuário não cadastrado.                    | Solicitação registrada e pendente de aprovação.       |
| UC03   | Visualizar todos Eventos e Cultos | Membro                             | Listar eventos disponíveis.                                 | Usuário autenticado.                       | Lista de eventos/cultos exibida.                      |
| UC04   | Visualizar tipo Eventos e Cultos  | Membro                             | Exibir categorias de eventos (Culto, Ensaio etc.).          | Eventos cadastrados.                       | Tipos exibidos conforme relacionamento na base.       |
| UC05   | Exibir detalhes de Eventos        | Membro                             | Mostrar informações detalhadas de um evento.                | Evento selecionado.                        | Detalhes exibidos (data, hora, tipo).                 |
| UC06   | Fazer Registro de Presença        | Membro                             | Permitir que o membro registre presença num evento.         | Evento em andamento e usuário autenticado. | Presença registrada (tabela `Presencas`).             |
| UC07   | Visualizar Post no Blog           | Membro                             | Acessar postagens públicas.                                 | Postagens existentes.                      | Postagens visíveis.                                   |
| UC08   | Manter Posts                      | Organizador                        | Criar, editar, excluir, arquivar posts.                     | Autenticado como organizador.              | Posts atualizados (tabela `Posts`).                   |
| UC09   | Exibir Perfil de Usuário          | Organizador                        | Visualizar perfil de usuário.                               | Usuário autenticado.                       | Dados exibidos conforme o perfil.                     |
| UC10   | Manter os Membros                 | Organizador                        | Cadastrar, editar ou excluir membros.                       | Autenticado como organizador.              | Tabela `Usuarios` atualizada.                         |
| UC11   | Manter os Eventos                 | Organizador                        | Adicionar, editar ou excluir eventos.                       | Autenticado como organizador.              | Tabela `Eventos` atualizada.                          |
| UC12   | Exibir Tipos de Eventos           | Organizador                        | Ver tipos de evento cadastrados (Culto, Ensaio).            | Tipos cadastrados.                         | Lista carregada da tabela `TipoEventos`.              |
| UC13   | Exibir os Perfis                  | Organizador                        | Ver tipos de perfis disponíveis no sistema.                 | Perfis definidos.                          | Lista carregada da tabela `Perfis`.                   |
| UC14   | Manter os Organizadores           | Administrador                      | Gerenciar contas de organizadores.                          | Autenticado como admin.                    | Tabela `Usuarios` atualizada com role de organizador. |
| UC15   | Gerar Relatório                   | Administrador                      | Criar relatórios de eventos, presença ou usuários.          | Dados disponíveis no sistema.              | Relatório gerado (baseado em `Eventos`, `Presencas`). |
| UC16   | Exportar Dados                    | Administrador                      | Exportar relatórios em PDF.                                 | Relatório gerado.                          | Arquivo PDF disponível.                               |
| UC17   | Visualizar Gráficos               | Administrador                      | Analisar dados estatísticos através de gráficos.            | Dados estatísticos computados.             | Gráficos visuais apresentados.                        |
| UC18   | Dar e Gerenciar Permissões        | Administrador                      | Alterar permissões de usuário (Membro, Organizador, Admin). | Acesso administrativo.                     | Perfil atualizado na tabela `Usuarios`.               |
