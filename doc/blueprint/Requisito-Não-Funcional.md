# REQUISITO NÃO FUNCIONAL (RNF)

- **RNF01**
        - **Nome**: Interface web responsiva.
        - **Descrição**: A interface do sistema será responsiva, adaptando-se a diferentes tamanhos de tela, incluindo dispositivos móveis.
        - **Categoria**: Usabilidade.

---

- **RNF02**
        - **Nome**: Suporte a autenticação e autorização por tipo de usuário.
        - **Descrição**: O sistema suportará autenticação de usuários, com autorização baseada no perfil (Administrador, Organizador, Membro).
        - **Categoria**: Segurança.

---

- **RNF03**
        - **Nome**: Segurança dos dados (criptografia de senhas, controle de acesso).
        - **Descrição**: O sistema assegurará a segurança dos dados dos usuários, com criptografia de senhas e controle de acesso rigoroso às informações sensíveis.
        - **Categoria**: Segurança.

---

- **RNF04**
        - **Nome**: Sistema leve, otimizado para celulares.
        - **Descrição**: O sistema será otimizado para garantir um bom desempenho, especialmente em dispositivos móveis com recursos limitados.
        - **Categoria**: Desempenho.

---

- **RNF05**
        - **Nome**: Notificações automáticas programadas.
        - **Descrição**: O sistema enviará notificações automáticas para os usuários, com base em eventos programados, sem necessidade de intervenção manual.
        - **Categoria**: Funcionalidade.

---
